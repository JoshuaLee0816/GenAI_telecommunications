# Network Troubleshooting Procedures

## Common Issues
1. Cannot connect to network
2. Slow internet speed
3. Frequent call drops
4. Wi-Fi disconnects

## Step-by-Step Resolution
1. Verify customer account and service subscription
2. Check base station / router status
3. Restart device remotely if possible
4. Schedule technician visit if unresolved

## Escalation
- If outage affects >10 customers, escalate to Level 2
- If repeated complaint within 24 hours, escalate to Level 2
